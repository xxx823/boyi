# 实现计划：国际跳棋AI程序

## 概述

本实现计划将国际跳棋AI程序的设计转化为可执行的编码任务。系统采用C++17实现，使用位棋盘（Bitboard）数据结构和Alpha-Beta剪枝搜索算法。实现分为7个主要阶段，从核心数据结构到高级功能和文档准备。

**技术栈：** C++17, Visual Studio 2022, CMake
**现有代码：** boyi/boyi.cpp（已实现基础Board类）
