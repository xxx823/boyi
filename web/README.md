# 国际跳棋 Web 界面

这是一个用于与国际跳棋AI对弈的Web界面。

## 功能特点

✅ **完整的游戏界面**
- 10×10国际跳棋棋盘
- 可视化棋子和走法
- 鼠标点击下棋

✅ **游戏功能**
- 新游戏
- 悔棋
- 走法提示
- 走法历史记录

✅ **AI对弈**
- 可选择玩家颜色（黑方/白方）
- 可调整AI难度（1-10秒思考时间）
- 显示AI思考过程

✅ **游戏规则**
- 强制吃子规则
- 最大吃子规则
- 升王规则
- 胜负判定

## 使用方法

### 方法1：直接打开（推荐）

1. 双击 `index.html` 文件
2. 在浏览器中打开即可开始游戏

### 方法2：使用本地服务器

如果需要连接C++后端，需要启动本地服务器：

```bash
# 使用Python启动简单HTTP服务器
cd web
python -m http.server 8000

# 或使用Node.js
npx http-server -p 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 游戏说明

### 基本操作

1. **选择棋子**：点击己方棋子
2. **移动棋子**：点击高亮的目标格子
3. **吃子**：如果有吃子机会，必须吃子（强制吃子规则）

### 游戏设置

- **玩家颜色**：选择执黑（先手）或执白（后手）
- **AI难度**：
  - 简单：1秒思考时间
  - 中等：3秒思考时间
  - 困难：5秒思考时间
  - 专家：10秒思考时间

### 游戏规则

1. **移动规则**
   - 普通棋子只能向前斜走
   - 王棋可以向任意斜方向移动

2. **吃子规则**
   - 跳过对方棋子进行吃子
   - 有吃子机会时必须吃子（强制吃子）
   - 必须选择吃子最多的走法（最大吃子）
   - 可以连续跳吃

3. **升王规则**
   - 黑方棋子到达第10行升王
   - 白方棋子到达第1行升王
   - 王棋用♔标记

4. **胜负判定**
   - 吃光对方所有棋子
   - 对方无合法走法

## 连接C++后端

要连接真正的AI引擎，需要：

1. **创建HTTP服务器**（使用C++或Node.js）
2. **实现API接口**：
   - `POST /api/move` - 获取AI走法
   - `POST /api/evaluate` - 评估局面
   - `GET /api/hint` - 获取提示

3. **修改 checkers.js**：
   ```javascript
   async aiMove() {
       const response = await fetch('/api/move', {
           method: 'POST',
           headers: { 'Content-Type': 'application/json' },
           body: JSON.stringify({
               board: this.game.board,
               player: this.game.currentPlayer,
               timeLimit: this.game.aiLevel * 1000
           })
       });
       const data = await response.json();
       // 使用返回的走法
   }
   ```
