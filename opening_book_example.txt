# 国际跳棋开局库示例文件
# 本文件展示开局库支持的所有格式

# ========== 文件格式说明 ==========
# 1. 注释行：以#开头的行会被忽略
# 2. 空行：用于分隔不同类型的开局，会被忽略
# 3. 走法格式：
#    - 普通移动：from-to（例如：6-11表示从6号格子移动到11号格子）
#    - 跳吃：fromxto（例如：15x24表示从15号格子跳吃到24号格子）
#    - 连续跳吃：fromxmid1xmid2xto（例如：15x24x33表示连续跳吃）
# 4. 走法序列：每行一个开局线，走法用空格分隔

# ========== 基础示例 ==========

# 示例1：简单的开局线（5步）
6-11 31-26 11-16 26-21 16-20

# 示例2：稍长的开局线（8步）
7-12 31-26 12-17 26-21 17-22 21-18 11-16 18-14

# 示例3：包含跳吃的开局线
8-13 31-26 13-18 26-21 18-23 21-17 11-16 17x10

# ========== 使用说明 ==========
# 
# 1. 如何使用本文件：
#    - 将本文件重命名为 opening_book.txt
#    - 放在程序可执行文件的同一目录下
#    - 程序启动时会自动加载
#
# 2. 如何添加新的开局：
#    - 在文件末尾添加新的开局线
#    - 每行一个开局序列
#    - 使用空格分隔走法
#    - 可以添加注释说明开局类型
#
# 3. 走法编号说明：
#    - 棋盘格子编号从1到50
#    - 只有黑色格子有编号
#    - 编号从左上角开始，从左到右，从上到下
#    - 第1行：1-5
#    - 第2行：6-10
#    - 第3行：11-15
#    - ...
#    - 第10行：46-50
#
# 4. 注意事项：
#    - 确保走法合法（程序会验证）
#    - 非法走法会被跳过并输出警告
#    - 建议每个开局线不超过15步
#    - 可以为同一初始局面添加多个变化
#
