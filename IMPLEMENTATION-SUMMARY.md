## 📝 使用说明

### 性能基准测试

**目的**: 测试AI引擎的各项性能指标

**运行**:
```bash
run-performance-tests.bat
```

**测试项目**:
1. 走法生成速度（目标：< 10μs）
2. 评估函数速度（目标：< 1μs）
3. 搜索速度NPS（目标：> 100K）
4. 置换表命中率（目标：> 50%）

**结果分析**:
- ✓ 表示达标
- ○ 表示接近目标
- ✗ 表示需要优化

### 自我对弈测试

**目的**: 验证AI的稳定性和平衡性

**运行**:
```bash
# 默认配置
run-selfplay.bat

# 快速测试
selfplay.exe 10 1000

# 自定义配置
selfplay.exe [对局数] [每步时间ms]
```

**参数说明**:
- 对局数：建议10-100局
- 每步时间：建议500-2000ms

**结果文件**:
- `selfplay_results.txt`：详细的对局记录

**分析指标**:
- 黑白胜率差异（应 < 10%）
- 和棋率（应 < 30%）
- 平均步数（应 30-80步）
- 无规则违规
- 无程序崩溃
